### Usage

To run the project:

  Clone the file into the desire foler:
  ```sh
    git clone https://github.com/popey17/Laravel-Api.git
    cd to the folder
  ```
  Run composer install
  ```
    composer install
  ```
  Create env file

  ```
    cp .env.example .env
  ```

  Generate application key
  ```
    php artisan key:generate
  ```

  Run database migration 
  ```
    php artisan migrate
  ```

 Start development server
 ```
    php artisan serve
 ```
